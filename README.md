### 프로젝트 소개 

`TeenTalk`은 비윤리적 메시지를 실시간으로 탐지하여 경고 알림을 보내는 메신저입니다. 

SNS상에서 이루어지는 청소년들의 언어폭력을 사전에 방지하고 언어 습관을 개선하기 위하여 `TeenTalk`을 구상했습니다.
